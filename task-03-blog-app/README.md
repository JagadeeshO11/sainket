# Task 03 — Simple Blog App

## Stack
Frontend only — HTML + CSS + JS + LocalStorage (no backend, no database)

## Structure
```
frontend/
├── index.html         → Blog UI
├── css/style.css      → Styling
├── js/storage.js      → LocalStorage CRUD (getPosts, addPost, deletePost)
└── js/app.js          → DOM rendering and events
```

## Features
- Create posts (title + content)
- View all posts (newest first)
- Delete any post
- Data persists in browser localStorage

## Run
Open `frontend/index.html` directly in browser. No server needed.

## Deploy
Frontend → Netlify / Vercel (drag and drop)
