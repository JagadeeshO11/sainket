# Task 04 — Number Guessing Game

## Stack
Frontend only — HTML + CSS + JS (no backend, no database)

## Structure
```
frontend/
├── index.html         → Game UI
├── css/style.css      → Styling
├── js/game.js         → Pure game logic (random number, guess check)
└── js/app.js          → DOM events, rendering, history
```

## Features
- Random number 1–100 each game
- Too High / Too Low hints
- Attempt counter
- Guess history log
- Play Again without page reload

## Run
Open `frontend/index.html` directly in browser. No server needed.

## Deploy
Frontend → Netlify / Vercel
