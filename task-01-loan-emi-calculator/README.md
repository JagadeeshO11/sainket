# Task 01 — Loan EMI Calculator

## Stack
Frontend only — HTML + CSS + JS (no backend, no database)

## Structure
```
frontend/
├── index.html       → UI form
├── css/style.css    → Styling
├── js/emi.js        → EMI formula, validation, formatting
└── js/app.js        → DOM events and rendering
```

## Formula
EMI = P × R × (1+R)^N / ((1+R)^N - 1)

## Run
Just open `frontend/index.html` in any browser. No server needed.

## Deploy
Frontend → Netlify / Vercel (drag and drop frontend folder)
